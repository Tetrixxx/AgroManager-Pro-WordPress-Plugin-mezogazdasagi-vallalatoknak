Komplex mezőgazdasági vállalatirányítási WordPress plugin.

== Description ==

Az AgroManager Pro egy komplex WordPress plugin, amelyet országos szintű, mezőgazdálkodással foglalkozó cégek számára fejlesztettünk. A plugin lehetővé teszi a teljes gazdasági tevékenység nyomon követését egyetlen felületen.

**Főbb funkciók:**

* 🗺️ **Földterület-kezelés** – Parcellák nyilvántartása, GPS koordináták, talajminőség
* 🌱 **Növénykultúra-nyilvántartás** – Vetési tervek, hozam követés, parcellánkénti bontás
* 🚜 **Géppark-kezelés** – Gépek állapota, szerviz nyilvántartás, üzemóra követés
* 🌤️ **Időjárás-integráció** – Open-Meteo API, 7 napos előrejelzés, parcella alapú lekérdezés
* 💰 **Pénzügyi modul** – Bevétel/kiadás nyilvántartás, éves összesítők, Chart.js diagramok
* 👷 **Dolgozó-kezelés** – Munkatársak és munkaidő nyilvántartás
* 📊 **Dashboard** – KPI áttekintés, pénzügyi grafikon, időjárás widget, szerviz figyelmeztetések

== Installation ==

1. Töltsd fel a plugin mappát a `/wp-content/plugins/` könyvtárba
2. Aktiváld a plugint a WordPress admin felületén
3. Keresd az „🌾 AgroManager" menüpontot az admin oldalsávban
4. Állítsd be az alapértelmezett helyszínt és pénznemet a Beállítások menüben
